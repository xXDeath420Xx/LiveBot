7. USAGE QUICK START
================================================================================

BASIC USAGE:

// Show loading overlay
loadingManager.showPageLoading('Loading...');

// Show form loading
loadingManager.showFormLoading(form);

// Show toast notification
loadingManager.showToast('Success!', 'success');

// Show data loading indicator
loadingManager.showDataLoading(container, 'Loading data...');

// API call with loading
loadingManager.fetchWithLoading('/api/data', {}, container);

// Wrap async operation
loadingManager.withLoading(asyncFn, container, 'Processing...');

See LOADING_STATES_GUIDE.md for complete documentation.

8. INTEGRATION INSTRUCTIONS
================================================================================

TO INTEGRATE INTO OTHER PAGES:

1. Add CSS link in <head>:
   <link rel="stylesheet" href="/css/loading-states.css">

2. Add partial include (after header):
   <%- include('partials/loading') %>

3. Add script before closing </body>:
   <script src="/js/loading-manager.js"></script>

4. Use in your scripts:
   loadingManager.showPageLoading('Loading...');

That's it! Full integration in 4 steps.
